## 简介

简单的仿淘宝热卖(素材来自于淘宝)

vue2 + element-ui(组件库) + axios(http库) + dayjs(处理时间库) + vue-piczoom(放大镜)

## 步骤

npm install  (安装依赖)

npm run serve  (运行)

npm run build  （打包）
